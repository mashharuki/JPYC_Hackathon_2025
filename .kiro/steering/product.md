# Product Overview

**Innocence Ledger（イノセンス・レジャー）** - ブロックチェーンとステーブルコインを活用した透明性のある冤罪支援プラットフォーム

冤罪被害者および無罪を主張する被収容者に対し、透明性・追跡可能性を担保した信頼できる資金支援インフラを提供します。

## Core Capabilities

- **透明な資金フロー**: ブロックチェーン上で支援金の流れを完全可視化
- **匿名性とプライバシー保護**: ZKP（ゼロ知識証明）を活用し、支援者・被支援者の匿名性を選択可能に
- **グローバル送金**: ステーブルコインによる国境を越えた支援を実現
- **ケース別プロジェクト管理**: 再審請求・弁護士費用・鑑定費用など用途別に支援を管理
- **既存団体との共存**: 冤罪支援団体の実行を支えるインフラとして機能

## Target Use Cases

### 支援を受ける側

- **無罪確定後の冤罪被害者**: 生活費・社会復帰支援
- **再審請求中の被収容者**: 弁護士費用・DNA鑑定費用・証拠収集資金

### 支援する側

- 個人支援者（国内・海外）
- 親族・市民
- NGO/NPO/財団
- 企業（CSR・寄付枠）

## Value Proposition

従来の寄付は「信頼」に依存していたが、本プロジェクトは「可視化された履歴」で判断可能に。

- **支援者**: 資金が確実に本人に届いたことをオンチェーンで確認
- **被支援者**: 継続的かつ透明性のある支援を受けられる
- **冤罪支援団体**: 自団体の活動をブロックチェーン化し、寄付者への説明責任を強化

## Project Philosophy

> 「善意は、どうすれば信頼できる形で届くのか？」

この問いに、ブロックチェーン技術で答えを出す。

## MVP Scope（ハッカソン）

- ケース1件の支援ページ
- ステーブルコイン送金機能
- オンチェーン履歴表示
- 資金用途ステータス表示

**MVPでやらないこと**: 法的判断、冤罪の事実認定、実運用での資金管理
