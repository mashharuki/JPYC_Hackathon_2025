# Innocence Ledger（イノセンス・レジャー）

**〜信頼を、証明に変える。冤罪被害者の再起を支えるオンチェーン支援インフラ〜**

## 1. Product Overview

Innocence Ledgerは、ブロックチェーンとステーブルコインに加えてマルチシグとゼロ知識証明を活用し、冤罪被害者および無罪を主張する被収容者に対し、**「検証可能な透明性」**と **「プライバシー」** を両立させた支援インフラを提供します。

既存の「組織への信頼」に依存した寄付モデルを、**「コードによる証明（Don't Trust, Verify）」** にアップデートし、日本、そして世界中の善意を最も必要としている人へ安全に届けます。

## 2. Core Capabilities & Tech Stack

本プロジェクトは、Ethereumエコシステムの最先端技術を司法支援に統合します。

- **Censorship-Resistant Support (ERC-4337 / AA / MultiSig Wallet)**:
  被収容者が身体的・経済的に隔離されていても、事前に設定したガーディアン（家族・弁護士）がマルチシグで支援金（JPYC）を運用。権利の執行を止めません。
- **Privacy via ZKP (Semaphore v4)**:
  支援者は社会的レピュテーションリスクから保護され、匿名性を保ちながら「正当な支援グループの一員であること」を証明し、寄付を行えます。
- **Programmable Transparency**:
  スマートコントラクトにより、資金の使途を「認証済み弁護士」や「科学鑑定機関」のアドレスに限定。不正流用が数学的に不可能な仕組みを構築します。
- **Global Liquidity**:
  JPYC等のステーブルコインを活用し、国境を越えた瞬時の支援受領と、Amazonギフト券等を通じた迅速な生活物資への転換を実現します。

## 3. Target Use Cases

### 支援を受ける側

- **再審請求中の被収容者**: 弁護士費用、DNA再鑑定費用、証拠収集のための活動資金。
- **無罪確定後の冤罪被害者**: 社会的信用を失った状態での当面の生活費、職業訓練、社会復帰支援。

### 支援する側

- **個人・海外支援者**: 透明性を重視し、ダイレクトに被害者を助けたい有志。
- **企業・財団**: CSR活動の一環として、使途が100%可視化された寄付を求める組織。

## 4. Value Proposition

| 観点           | 従来の寄付（Web2）       | Innocence Ledger（Web3）               |
| :------------- | :----------------------- | :------------------------------------- |
| **信頼の対象** | 支援団体の管理・報告     | **スマートコントラクトと台帳**         |
| **透明性**     | 報告書を待つ必要がある   | **リアルタイムでオンチェーン検証可能** |
| **匿名性**     | 銀行振込等で特定のリスク | **ZKPによりプライバシーを完全保護**    |
| **送金速度**   | 銀行の営業時間に依存     | **24時間365日、秒速で着金**            |

## 5. Project Philosophy

> **"Rights are not just words. Rights are code execution."**
> （権利とは単なる言葉ではない。権利とは、コードの執行である。）

私たちは、日本の司法が抱える「99.9%」という絶望的な有罪率の裏側にある、個人の無力化というシステムエラーを解決します。

司法の不完全さを嘆くのではなく、テクノロジーによって「個人の防御権の可用性」を保証します。

## 6. MVP Scope（Hackathon Focus）

48時間のハッカソンにおいて、以下のコア機能を実証します。

- **Case Dashboard**:
  支援を必要とするケースの概要と、リアルタイムの資金調達状況表示。
- **Trustless Donation**:
  Semaphoreを用いた匿名寄付、およびJPYCの送金フロー。
- **On-chain Traceability**:
  支援金がどのようにプールされ、用途（Status）が更新されるかの履歴表示。
- **Evidence Attestation**:
  弁護士による活動報告や証拠ハッシュのオンチェーン記録（EASの簡易実装）。

**Out of Scope**:
独自の法的判断、冤罪の事実認定アルゴリズムの実装（外部の専門家との連携を前提とするため）。
